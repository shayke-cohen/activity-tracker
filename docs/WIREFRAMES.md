## Watch Face Complications

### Circular Complication (Steps Progress)

```
    On Watch Face:
    
         ╭───╮
        ╱     ╲
       │ 12.4K │   ← Steps count
       │  👟   │   ← Icon
        ╲     ╱
         ╰───╯
           ▓▓▓▓▓▓▓░░  ← Progress ring (124% filled)
```

### Corner Complication (Calories)

```
    Watch Face Corner:
    
    ┌─────────────────┐
    │                 │
    │            520  │ ← Value
    │            🔥   │ ← Icon
    │           ▓▓▓░  │ ← Mini gauge
    │                 │
```

### Rectangular Complication (Multi-Metric)

```
    On Watch Face:
    
    ┌─────────────────┐
    │ 👟 12,456 steps │
    │ 🔥 520 cal      │
    │ 📏 5.2 km       │
    └─────────────────┘
```

### Inline Complication (Text Only)

```
    Watch Face (single line):
    
    ─────────────────────────
    👟 12,456 steps • 124%
    ─────────────────────────
```

### Active Workout Complication

```
    When workout is active:
    
         ╭───╮
        ╱     ╲
       │ 32:45 │   ← Duration
       │  🏃   │   ← Activity icon
        ╲     ╱
         ╰───╯
         
    Tap to open workout view
```
